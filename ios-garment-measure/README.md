# GarmentMeasure iOS Prototype

独立 iOS 原生原型，不依赖当前仓库里的 FastAPI 或微信小程序。

## 功能

- 使用 ARKit 在水平平面上点选两点测距。
- 将最近一次测距保存为 T 恤尺寸字段。
- 根据已保存尺寸生成一个可旋转的 SceneKit 3D T 恤比例预览。

## 运行

1. 用 Xcode 打开 `GarmentMeasure.xcodeproj`。
2. 选择真实 iPhone 设备运行。ARKit 相机测距不能在模拟器里实际验证。
3. 将衣服平铺在桌面或地面，尽量让手机与衣服平面保持稳定。
4. 在「测量」页选择尺寸字段，点选衣服上两个关键点，点击确认保存。
5. 切换到「3D」页查看参数化预览。

## 当前边界

- 这是 AR 尺寸采集 + 参数化建模原型，不是照片级 3D 扫描。
- 测量精度受平面识别、光线、相机角度、衣服是否铺平影响。
- 当前只做 T 恤模板，后续可以按同样的数据结构增加裤子、裙子、外套。
